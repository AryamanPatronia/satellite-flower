Step 1: Run setup.sh using "./setup.sh"...

Running this will install create and activate the virtual environment.
It will install all the dependencies using the requirements.txt file.
It will also download the data and partition it.
This is an essential step to do before the running the simulations.

Step 2: Make sure that docker is running in the background before you start the simulation.


Step 3: Navigate to the root folder and run "./start_all.sh"...

Right now this setup is ONLY for MacOS users.
